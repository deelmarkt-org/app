# ASO Keyword Research — DeelMarkt

> Evidence file for keyword selection. Update after each ranking cycle (monthly).
> Source: Sensor Tower / AppFollow free tier + manual brainstorm (2026-04-15).

---

## Methodology

1. Seed list from product brief: tweedehands, marktplaats alternatief, veilig verkopen,
   escrow app nederland, p2p marktplaats, gebruikte spullen verkopen, iDIN verificatie,
   dutch marketplace.
2. Expanded via App Store "Customers also searched for" on Marktplaats + Vinted (NL store).
3. Difficulty scored 1–10 (10 = highest competition) from Sensor Tower free-tier estimates.
4. DeelMarkt relevance scored 1–5 (5 = core differentiator).
5. Target: top 20 NL keywords (primary market) + top 20 EN keywords.
